# 🚀 AutoAgent-MCP: Automated Meta Ads & GA4 Sync Agent

## 소개
이 프로젝트는 **Model Context Protocol(MCP)**을 기반으로 설계된 AI 에이전트 자동화 도구입니다. 
마케터와 개발자가 AI 솔루션(예: Claude Code, OpenClaw 등)을 통해 복잡한 코드 작성 없이도 **Meta Ads API 제어, 네이버 데이터 크롤링, 그리고 Google Analytics 4(GA4) 및 Google Tag Manager(GTM)의 태깅 및 데이터 추적을 자동화**할 수 있도록 연결해 주는 오픈소스 브릿지 역할을 합니다.

## 주요 기능
* **MCP 기반 AI 연동**: Claude, Cline 등 최신 AI 코딩 환경에서 Meta Ads 및 웹 데이터에 직접 접근할 수 있는 MCP 서버 기능 제공
* **마케팅 자동화 파이프라인**: 광고 캠페인 성과 데이터를 실시간으로 수집하고, 맞춤형 ROAS 지표를 계산하여 대시보드로 전송
* **크롤링 및 데이터 트래킹**: 네이버 블로그/검색 광고 데이터 크롤러 내장 및 GTM 맞춤 설정 자동 생성 기능 지원
